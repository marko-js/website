Supported Environments

Browser

Marko supports all versions of browsers that are also supported by eBay. This list is publicly available on browserslist-config, and mirrored here.

[
  "Chrome >= 109",
  "Firefox >= 115",
  "Safari >= 15.4",
  "Edge >= 145",
  "Opera >= 129",
  "ChromeAndroid >= 136",
  "ios_saf >= 15.4"
]
View in browsersl.ist

Node

We support all versions of node that are maintained, aligning with the official node support release schedule.
