# Newsletter

The Marko newsletter is a monthly summary of shipped work, written for the developers who build with it. Each issue gathers the month's features, fixes, performance gains, and tooling improvements into a short read, with links to the pull requests behind every change.

> [!TIP]
> New issues are published to an [RSS feed](/docs/newsletter/feed.xml). Add the feed URL to any reader to follow along.
