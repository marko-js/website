# Supported Environments

## Browser

Marko supports all versions of browsers that are also supported by eBay. This list is publicly available on [browserslist-config](https://github.com/eBay/browserslist-config/blob/master/index.js), and mirrored here.

## Node

We support all versions of node that are maintained, aligning with the official [node support](https://nodejs.org/en/about/previous-releases) release schedule.
